DATABAE
----------------------

a simple localStorage abstraction. There are many like it but this one is mine.

-----------------------------

auto JSON.(stringify/parse)'s stuff so you don't have to care about squat

### general
- get(key)
- set(key, val)

### numeric
- inc(key, amt || 1)  
- dec(key, amt || 1)

### arrays
- push(key, el)
- remove(key, el)
